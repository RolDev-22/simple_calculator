# simple_calculator
Calculadora sencilla para practica de conceptos básicos del uso de JavaScript, se emplean funciones de uso común y una interfaz sencillacon estructura de HTML y estilo de CSS
se realizan funciones de suma, resta, multiplicación, división, porcentage, raíz cuadrada, captura de datos, uso de variables auxiliares.
Se realiza una interfaz amigable, moderna y responsive
